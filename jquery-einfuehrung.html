jQuery ist die Bibliothek, um dynamische Webseiten zu programmieren. Besonders die einfache API und die komfortable Plugin-Architektur sorgen dafür, dass die Bibliothek das Mittel der Wahl ist. Je nach Schätzung nutzen 50 bis 60% aller Webseiten weltweit jQuery.


Der Kurs richtet sich vor allem an Einsteiger, die lernen wollen, wie man dynamische Webseiten erzeugt. Vorkenntnisse in HTML und CSS genügen um direkt den Einstig zu machen. Javascript-Kenntnisse sind für diesen Kurs nicht erforderlich.


Der Kurs umfasst 7 Kapitel (+ Bonuskapitel), in denen man lernen kann, wie jQuery Elemente auf der Seite über die so genannte „selector engine“ finden kann und wie man Elemente verändert. Im Folgenden wird erklärt, wie man mit jQuery einfach Animationen programmiert. Danach folgt ein Kapitel über jQuerys Event-System und dessen Handhabung an einem praktischen Beispiel. Im Bonuskapitel wird an einem konkreten Beispiel erklärt, wie man ein bestehendes Plugin verwendet und konfiguriert. Das letzte Kapitel umfasst, wie man sich im Internet Hilfe und weitere Anleitungen beschafft, falls man an einer Stelle hängen bleibt.
