KeineInternetseite kommt ohne HTML & CSS aus und deshalb lohnt es sich, beide zusammen zu lernen.

In diesem Kurs wollen wir uns nicht nur die Basics, wie Textformatierung, Farben, Links und Bilder anschauen, sondern auch lernen, wie man moderne Webseiten mit barrierearmem HTML und CSS umsetzt, ohne auf Tabellen oder ähnliche stützen zurückgreifen zu müssen. Mit Abschluss des Kurses wirst du HTML & CSS verstanden haben, kannst Elemente auf der Seite positionieren und bist in der Lage sein Basislayouts in Code umzusetzen. Du hast außerdem fundiertes Wissen, um dich auf Seiten wie selfhtml zurechtzufinden und selbstständig deine Fähigkeiten auszubauen.
